## 日程计划工具  

该项目是我2019年春季课余时间做的一个提醒自己的小工具，含有

- 每日日程\每周日程\每月日程
- 3个月计划\1年计划\5年计划
- 设置自己的目标和目前状态
- 日记功能

### 执行文件

工具是用MFC做的，所以只支持Windows界面，可在win7/10上正常运行

| 软件名称 | 日程计划软件           |
| -------- | ---------------------- |
| 软件位置 | release/Schedule.exe   |
| 软件大小 | 157KB                  |
| 初始密码 | 用户名:root 密码:admin |
| 支持平台 | win7/win10             |
